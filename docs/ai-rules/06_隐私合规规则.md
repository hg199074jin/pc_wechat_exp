# 隐私合规规则

## 数据上传边界

- 聊天数据**完全在本地**，不上传互联网
- **调用 AI 功能时**，所选聊天记录的**文本内容**会发送给配置的 LLM 服务
- **媒体文件（图片/语音/视频）不上传**
- 调用前需知悉并自行评估 LLM 服务的数据合规性

## 敏感信息

- API key 在日志/错误信息/调试文件中**不得泄漏**
- `call_llm` 的错误信息**不透传**上游完整响应体到前端（避免 key 片段/乱码）
- `_write_artifact_json_debug` 落盘内容**不含 api_key**
- 调试文件含群聊原文（敏感），应有清理机制（待 Phase 4）

## 提示词注入防护

- LLM 输出（分析报告/卡片正文）渲染到前端前必须清洗（剥离 script/on*/javascript:）
- 被分析的群聊内容可能含恶意 prompt injection

## 法规

- 中国境内未经授权的个人信息收集和处理违反《个人信息保护法》
- 工具仅供个人数据使用，禁止用于他人数据

## 不要做

- 不上传媒体
- 不把 api_key 写进调试文件/日志/前端
- 不在未提示用户的情况下发送数据到外部
- 不声称"完全匿名化"（需单独验证的脱敏设计）
